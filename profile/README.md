# Staros Labs

Staros Labs builds personal infrastructure, self-hosted workflows,
and operational software used in daily life.

## Focus Areas

- **Infrastructure and automation** for self-hosted services,
  deployment workflows, and operational visibility
- **Personal software** for knowledge, home, and property management
- **Applied product experiments** that start internal
  and mature into maintained tools

## Public Repositories

- [`.github`](https://github.com/Staros-Labs/.github) -
  Organization profile and shared community health files
- [`node-hp-scan-to`](https://github.com/Staros-Labs/node-hp-scan-to) -
  HP scan-to-computer utility
- [`unraid-templates`](https://github.com/Staros-Labs/unraid-templates) -
  Unraid Community Applications templates
- [`Radarr`](https://github.com/Staros-Labs/Radarr) -
  Maintained fork for local infrastructure use
- [`Sonarr`](https://github.com/Staros-Labs/Sonarr) -
  Maintained fork for local infrastructure use

## Private Work

Most active product, infrastructure,
and operations repositories are private. That includes:

- customer and product applications
- internal infrastructure automation
- deployment runbooks and operational documentation

Organization members should use the member profile view
and internal handbook for onboarding and navigation.
